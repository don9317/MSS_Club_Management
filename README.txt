MSS Club Management Suite Demo v0.17

WORKFLOW
Registration
→ Check-In / No-Show tracking
→ Court and evaluator assignment
→ Tablet or paper evaluation
→ Director review
→ Big Board placement
→ Parent communication
→ Parent accepts, declines, or requests information
→ Payment plan selection
→ Online or offline payment entry
→ Deposit / first payment verification
→ Active roster

GITHUB PAGES
1. Unzip this package.
2. Upload index.html and MSS_Tryout_Demo_150_Players.csv to the repository root.
3. In GitHub Settings > Pages, publish from the main branch and root folder.

PAYMENTS RESTORED IN v0.17
- MSS / Stripe payments
- ACH payments
- Cash
- Check
- Invoice
- External payment
- Scholarship / credit
- Refund
- Charge adjustment
- Payment CSV import
- Payment ledger export
- Reference or check number
- Applied-to category
- Entered-by field
- Notes
- Calculated player charges, payments, credits, refunds, adjustments, balance, and financial status
- Roster activation based on the assigned plan and actual deposit payments or credits

IMPORTANT
This standalone GitHub prototype stores data in one browser. It demonstrates the complete workflow but does not connect to Stripe, send messages, or share data among devices. The production MSS version should use a shared database, authenticated users, MSS/Stripe payment posting, and a parent payment portal.
