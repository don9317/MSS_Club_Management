MSS Club Management Suite Demo v0.26

FIXED
- Auto-Balance now respects the courts assigned to each tryout session.
- A 3rd-grade session using Courts 1 and 2 will distribute 3rd graders only to Courts 1 and 2.
- Assignment court dropdowns show only courts eligible for that player's session.
- Session-specific evaluators are used when entered; Setup evaluators remain the fallback.
- Adding the same session/grade/time again now merges the selected courts into one configured session rather than creating disconnected duplicates.
- Existing invalid court assignments are cleared when they are outside the player's session.
